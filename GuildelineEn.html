How to use Website Header

- In each section, when clicked, it will navigate to the corresponding page in the same language as the previous page. For example, if you are currently on the homepage in Thai, clicking 'Download' will take you to the Download page in Thai as well.

- In the 'TH' section, clicking it will switch the current page to Thai. For example, if you are on the English homepage and click 'TH', it will switch to the Thai homepage; if you are on the English Download page and click 'TH', it will switch to the Thai Download page.

- To return to the homepage, simply click on the logo.

How to Download File

When downloading files, go to the Download page, select the desired category, and click on the desired item. Clicking it will take you to the Google Drive page, where you can download the file directly from Google Drive.
